Introduction

HTML is a markup language that is used to define the structure of a website's content.

It uses elements, and these elements can be used in various ways to present a developer's website.

HTML includes various attributes to better define the role of an individual element.

Elements

An HTML element is structured with a starting tag, a closing tag, and the content within. For instance, <div>Hello World!</div> would produce Hello World!

Attributes

There are various attributes that can be used to give developers greater control over elements.

These attributes can help developers use CSS to style them, or to allow the user to interact with them in some way. Some of the more commonly used attributes include:

  • id - ids are unique to a specific element and cannot be used for another one. They allow the developer to style them specifically, or to link them to an anchor tag.
  • class - unlike ids, classes can be used for multiple elements. If the developer wants to style several similar elements, they may give them the same class.
  • href - the href attribute is used to create links between elements or websites. If the user clicks that element, they will be transported to the target destination.
Semantic HTML

In order to make the structuring and readability of HTML easier, it is good practice to use Semantic HTML.

Semantic HTML refers to the meaning of a piece of code. Where one could create a main heading element using a simple div, the h1 element is a clearer indication of what you are trying to do.

Semantic HTML also allows for some automatic styling. An h1 element will be given a larger font to make it look like a heading. These are some commonly used semantic elements:

  • h1 through h6 - The heading elements allow for the developer to specify the importance and position of each heading, with h1 being the most important, and h6 the least. <h1>Hello World!</h1>
  • p - The paragraph element allows for easier grouping of text. A paragraph element is a block-level element, meaning it starts on a new line.<p>Hello World!</p> would produce:
    Hello World!
